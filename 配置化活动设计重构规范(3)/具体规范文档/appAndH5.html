概述

本文主要介绍配置化活动设计重构规范（v1.0）具体事项，凡涉及到图片更新事项，请参考以下规范标准。

图片优化规范
  1. 图片必须压缩，一般 80% 品质即可，保证文字清晰；
  2. 图片必须使用 渐进式图片 ：使用Photoshop的“存储为web所用格式”时候，勾选“连续”；
  3. 图片必须使用压缩工具压缩后上传；
  4. 图片体积不要超过 300 k；
注意事项
  1. 字体加粗蓝色部分为重构需要注意事项，仅供参考。
  2. 雪碧图根据功能需要，自行生成。目前最全的雪碧图按钮包含以下几个按钮：头图三按钮（即下载游戏、下载心悦APP、备用按钮 202*69），新手与特权的立即领取按钮与置灰按钮（238*71），充值豪礼包立即领取与置灰按钮（113*40）。
  3. 不允许修改及删除样式名。
  4. 理论上不支持添加background-image: url()属性。

顶部 banner

  1. 图片尺寸大小 750 * 880 px；
  2. 图片格式 jpg；
  3. 图片底部需留出 144px 的高度便于放置登录与活动信息文案。

顶部 banner 按钮

  1. 按钮背景是 202 * 69 px；
  2. 图片格式 png 雪碧图；
  3. 距离顶部 690px 。
  4. 最多可放置 3 个按钮。（只有一个或者两个按钮时候会整体居中。）
  5. 重构需要再可配置样式包内设置按钮的 background-position （因为按钮整合成一张雪碧图），对应样式类名： .wm-spr.h-spr-btn1, .wm-spr.h-spr-btn2, .wm-spr.h-spr-btn3

全局背景

  1. 图片宽度为 750px，高度视情况而定，玩法模块数量多，背景高度需要相对应增高。（背景高了，图片的体积相对应增加，在设计上可以斟酌考虑体现的手法）；
  2. 图片格式 jpg；
  3. 重构需要再可配置样式包内设置背景高度，使其正常显示。
    样式类名：.wm-section

新手礼包

  1. 设计礼包图片尺寸为 730px * 373px 。
  2. 按钮背景尺寸为 238px * 71px 。
  3. tab 最多支持五个选项，每个 tab 等比宽度。
  4. 重构可以对相对应的地方修改颜色风格：
    玩法标题：.wm-section .wm-title p
    玩法说明：.wm-section .wm-explain
    玩法tap选框：
        未选中：.wm-section .wm-nav li
        选中时：.wm-section .wm-nav li.current
    玩法tap选框：
        未选中：.wm-section .wm-nav li
        选中时：.wm-section .wm-nav li.current
    领取按钮：
        默认：.wm-spr.np-spr-btn1
        置灰：.wm-spr.np-spr-btn1.geted

充值礼包

  1. 礼包图片尺寸为 426px * 196px 。
  2. 按钮大小尺寸为 113px * 40px 。
  3. 图片格式为 png 雪碧图。
  4. 按钮样式类名：
    .wm-section .wm-recharge .rec-spr-btn2
    .wm-section .wm-recharge .rec-spr-btn2.geted

抽奖模块

  1. 抽奖图片尺寸可以调整。
  2. 图片尺寸可以 jpg 或者 png（png 为宜）。
  3. 抽奖奖品选中效果可以调整。
  4. 重构 dom 节点不可修改。
  5. 重构可以调整转盘旋转效果，可以依次打横行走，也可以围绕行走。
  6. 重构可调整 js 中 SWFOBJ 对象，来调整抽奖框按钮的大小。
  7. var lottery = new Lottery({
    'lighturl': 'ossweb-img/loterry-light.png',//外部光圈png 不填写就用默认的效果
    'starturl': '', //外部按钮png 不填写就用默认的按钮效果
    'width': 718,//转盘宽度
    'height': 557,//转盘高度
    'total': 9,//抽奖的奖品总数
    'sbtnx': 184,// 开始抽奖按钮的位置x坐标
    'sbtny': 560,// 开始抽奖按钮的位置y坐标
    'sbtnw': 320,// 开始抽奖按钮的宽度
    'sbtnh': 122,// 开始抽奖按钮的高度
    'boxw': 243,// 奖品光效的宽度
    'boxh': 193,//奖品光效的高度
    'position': "14_18_0,225_18_1,437_18_2,14_175_3,225_175_4,437_175_5,14_340_6,
                 225_340_7,437_340_8",
    //奖品光效的位置，对应奖品图片的布局，填入每个奖品的位置以及角度用逗号分割 x_y
    例如19_20或者19_20_0 表示第一个奖品的位置 x坐标为19px y坐标为20px 角度为0。）
    'contentId': 'lotterycontent',//嵌入lottery 的div层的 id
    'onClickRollEvent': callJsToStart,//对应上面接口
    'onCompleteRollEvent': callJsToComplete //对应上面接口
});

图片轮播模块

  1. 轮播图片大小无固定尺寸。但是所有图片尺寸大小必须一致。

活动规则

  1. 重构可修改框中的背景和边框，对应的类名：.wm-section .wm-rule。
  2. 重构修改规则中的字体大小、颜色，行间距，对应的类名：.wm-section .wm-rule li。
